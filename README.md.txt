# 代码仓库：MOF对盐碱土氮循环微生物的影响

## 描述
本仓库包含论文"Effects of microbial organic fertilizer on nitrogen cycling-related microbial communities and nitrogen-fixing system stability in saline soil"中使用的所有分析代码。

## 使用说明
1. 安装依赖：`install.packages(c("vegan", "igraph", "ggplot2"))` (R) 或 `pip install -r requirements.txt` (Python)
2. 按照顺序运行脚本：01 → 02 → 03 → 04
3. 主要输出保存在 `results/` 目录
